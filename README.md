# Bonk

- **install:** `curl -fsSL https://tailscale.com/install.sh | sh`
- **run:** `sudo tailscale up --advertise-exit-node --ssh`
- **get IP:** `sudo tailscale up --advertise-exit-node --ssh`
- **clone:** `git clone git@github.com:DanielFL0/bonk.git`
- **cd:** `cd bonk/`
- **cp:** `cp .env.example .env`
- **compose:** `docker compose up -d`
